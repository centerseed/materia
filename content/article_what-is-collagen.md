# 膠原蛋白是什麼？功效、吸收、怎麼補一次看懂

> 洗臉時捏一下臉頰，回彈速度好像沒有以前快；蹲下綁鞋帶,膝蓋偶爾會卡一下——這些變化,很多人第一個聯想到的都是「膠原蛋白流失了」。但膠原蛋白到底是什麼、為什麼會流失、吃下肚的膠原蛋白真的能被身體用嗎？這篇從最基礎的概念講起,幫你把膠原蛋白搞懂。

## 膠原蛋白是什麼、從哪來

膠原蛋白（Collagen）是人體含量最多的蛋白質,約佔全身蛋白質總量的三分之一。它是**結締組織的主要黏合物質**,像一張張細密的支架網,撐起皮膚真皮層（約 75% 由膠原蛋白構成）、骨骼、肌腱、韌帶、軟骨與血管。可以把它想成身體的「鋼筋」——皮膚有彈性、關節能靈活活動,都跟這張支架網夠不夠密實有關。

問題是,膠原蛋白從 **20 歲左右就開始加速流失**,合成速度追不上分解速度,年紀愈大差距愈明顯。60 歲後,皮膚真皮層厚度可能比年輕時降低 25-30%,這也是皮膚彈性下滑、關節活動度變差時常被提到的背景原因之一。

口服補充的膠原蛋白,來源不只一種。常見的是**水解魚膠原蛋白胜肽**（如日本 NIPPI® 採用魚鱗＋魚皮,經酵素水解製成平均分子量約 4000-5000Da 的小分子胜肽,人畜共通疾病風險也較牛豬來源低）;另一類是近年較新的**蛋殼膜**（如西班牙 OVODERM®,萃取自雞蛋殼內側薄膜,採專利水分離技術保留天然複合結構,富含第一型膠原蛋白、彈性蛋白與醣胺聚醣,是少數蛋素／植物飲食者仍能攝取的膠原蛋白來源）。不論來源,核心邏輯都一樣：把大分子膠原蛋白**水解或萃取成人體較容易利用的型態**,做為日常保養的營養補給。

## 有什麼好處

膠原蛋白相關的人體研究,目前累積證據較集中在兩個面向:

**皮膚彈性與保水。** 魚鱗膠原蛋白胜肽的人體外用試驗與細胞實驗顯示,能促進皮膚纖維母細胞增生、提升前膠原蛋白合成,皮膚相對彈性有觀察到提升;蛋殼膜的人體試驗則以客觀儀器與主觀量表追蹤,發現膚色均勻度與細紋滿意度有階段性改善,搭配細胞與動物實驗,觀察到能促進皮膚細胞分化與屏障基因表現、提升膠原蛋白與玻尿酸合成酶活性。這些研究支持膠原蛋白做為**養顏美容、維持皮膚彈性與保水**的日常保養選擇。

**關節日常保養。** 一篇獨立系統性回顧與統合分析（納入 4 篇 RCT、共 507 人）顯示,口服膠原蛋白胜肽對膝關節疼痛緩解達統計顯著;蛋殼膜也有三篇獨立人體隨機對照試驗支持,其中一篇劑量反應試驗發現,每日補充 500mg 一段時間後,疼痛自評量表改善幅度明顯優於安慰劑組。這些研究讓膠原蛋白逐漸被視為**中老年關節日常保養**的營養補給選項之一。

除此之外,蛋殼膜還有一篇以電腦化毛髮密度分析儀器追蹤的人體試驗,顯示補充 12 週後毛髮密度顯著優於安慰劑組,是目前少數有客觀儀器數據支持毛髮密度維持的口服原料之一。

## 為什麼會想額外補充？

日常三餐裡,真正富含完整膠原蛋白的食物其實不多——雞湯、豬腳、魚皮這類食物油脂負擔重,也不一定適合天天吃;而且**大分子的天然膠原蛋白不易被腸道直接吸收利用**,這也是市售產品多會強調「水解」「小分子胜肽」的原因,目的是把不易消化的大分子先拆解成人體較容易利用的型態。

另外,人體要**合成**自己的膠原蛋白,還需要維生素 C 作為必需輔因子——維生素 C 是水溶性營養素,容易受烹調與保存流失,現代人蔬果攝取量普遍不足,間接也影響體內膠原蛋白的合成效率。像 PureWay-C® 這類經人體研究驗證吸收速度較快的維生素 C 複方,常被拿來與膠原蛋白胜肽或蛋殼膜搭配補充,概念上就是「原料＋輔因子」一起顧。

## 怎麼吃

- **一般水解膠原蛋白胜肽**：原廠建議每日 2-3g 作為日常保養;相關人體研究測試劑量則落在每日 2-10g 不等,實際攝取量以產品標示為準。
- **蛋殼膜**：供應商建議每日 300mg;人體研究中關節保養面向測試範圍為每日 300-500mg,肌膚／毛髮面向多以每日 450mg 進行。台灣衛福部公告蛋殼膜每日食用限量為 500mg 以下,選購時留意單日總攝取量,避免多項產品疊加超標。
- **搭配維生素 C**：維生素 C 是人體合成膠原蛋白的必需輔因子,兩者一起補充是常見的日常保養組合,一般食品維生素 C 建議攝取量可參考產品標示,膠囊／錠狀食品之維生素 C 總含量台灣法規上限為每日 1000mg。
- **持續攝取比單次大劑量重要**：膠原蛋白相關的改善多是長期規律補充下觀察到的,建議固定每天吃,搭配正常作息,效果較容易被感受到。

## 誰適合、要注意什麼

膠原蛋白補充適合關節保養族群、熟齡肌膚保養族群、想由內而外保養的族群。若是蛋素／植物飲食者,或對海鮮／魚類過敏,蛋殼膜是少數仍可攝取的膠原蛋白選項；反過來說,對雞蛋過敏者則**禁止**食用蛋殼膜。

安全提醒方面:

- 對魚類海鮮過敏者,選購魚膠原蛋白胜肽產品前務必確認來源標示;對雞蛋過敏者則要避開蛋殼膜。
- 孕婦、哺乳期婦女、慢性病患者使用前建議諮詢醫療專業人員,現有多數研究對象為一般健康成人,尚未完整涵蓋特殊族群的安全性資料。
- 目前已發表的人體試驗中,未有嚴重不良事件回報,少數輕微不良反應也與產品無明確關聯,整體安全性資料尚屬正向,但仍建議依產品標示攝取,不需要一次補到超高劑量。

## 常見迷思 Q&A

**Q1：吃膠原蛋白會直接變成皮膚裡的膠原蛋白嗎？**
不完全是。口服的膠原蛋白胜肽進入腸道後,大多會先被分解成更小的胺基酸與短鏈胜肽再吸收,並不是「原封不動」搬到皮膚裡。研究觀察到的是,這些胜肽片段能刺激皮膚纖維母細胞、促進自身膠原蛋白合成酶的活性,理解成「提供原料＋刺激身體自己生產」會比較貼近實際機轉,而不是「吃進去直接補到臉上」。

**Q2：膠原蛋白補充品可以取代防曬、消除皺紋嗎？**
研究支持的是**皮膚彈性與保水的日常保養**,例如彈性儀讀數、細紋主觀滿意度的階段性改善,但這跟醫美等級的「除皺」「拉提」效果不是同一回事,也不能取代防曬——紫外線本身就是加速膠原蛋白分解的主因之一,日常防曬仍然是基本功。

**Q3：膠原蛋白種類這麼多,是不是分子量愈小愈好？**
分子量小、水溶性佳確實是水解胜肽設計時想達到的方向之一,但目前研究對「多小才是最佳」還沒有統一定論,不同來源（魚膠原胜肽 vs 蛋殼膜天然複合結構）本身的成分組成、伴隨的營養素也不一樣。與其只看分子量數字,更務實的做法是依照自己的需求（皮膚、關節、飲食限制如蛋素或海鮮過敏）挑選合適的來源。
